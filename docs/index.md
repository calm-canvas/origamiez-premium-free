# Origamiez Documentation

Welcome to the Origamiez WordPress theme documentation. This documentation is designed to help both developers and end-users make the most of the Origamiez theme.

## Available Documentation

### [Developer Guide](developer-guide.md)

The Developer Guide provides technical documentation for theme developers, including:

- Project structure and architecture
- Development environment setup
- Build process
- Theme customization
- Coding standards
- Plugin integrations
- Troubleshooting

### [User Guide](user-guide.md)

The User Guide provides end-user documentation for website administrators, including:

- Theme installation
- Theme setup and configuration
- Using the WordPress Customizer
- Working with widgets and menus
- Special features and templates
- Mobile responsiveness
- Plugin compatibility
- Troubleshooting
