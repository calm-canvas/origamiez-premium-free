# Origamiez WordPress Theme - User Guide

## Introduction

Welcome to Origamiez, a flexible and responsive WordPress theme designed for magazine, newspaper, and forum websites. This user guide will help you set up, customize, and make the most of your Origamiez theme.

## Installation

### Requirements

- WordPress 5.5 or higher
- PHP 7.4 or higher

### Installing the Theme

1. Log in to your WordPress admin panel
2. Go to **Appearance > Themes**
3. Click **Add New**
4. Click **Upload Theme**
5. Choose the Origamiez theme zip file
6. Click **Install Now**
7. After installation, click **Activate**

## Theme Setup

### Setting Up the Homepage

Origamiez offers multiple homepage layouts:

1. Go to **Pages > Add New**
2. Create a new page titled "Home"
3. Select one of the following templates from the Page Attributes section:
   - Default Template (standard blog layout)
   - Magazine Template (magazine-style layout)
   - Three Columns Template (three-column layout)
4. Publish the page
5. Go to **Settings > Reading**
6. Set "Your homepage displays" to "A static page"
7. Select your new "Home" page as the homepage
8. Click **Save Changes**

### Setting Up Menus

1. Go to **Appearance > Menus**
2. Create a new menu or edit an existing one
3. Add pages, categories, custom links, etc. to your menu
4. Under "Menu Settings," select one of the following locations:
   - Main Menu (primary navigation)
   - Top Menu (secondary navigation, does not support sub-menus)
   - Footer Menu (footer navigation, does not support sub-menus)
   - Mobile Menu (mobile navigation)
5. Click **Save Menu**

## Theme Customization

### Using the Customizer

1. Go to **Appearance > Customize**
2. Explore the following customization options:

#### Site Identity
- Upload your logo
- Set site title and tagline
- Upload a favicon

#### Colors
- Choose from predefined color schemes
- Customize individual color elements

#### Header Options
- Configure header layout
- Set header image
- Adjust header spacing

#### Footer Options
- Configure footer layout
- Customize footer text
- Set footer widgets display

#### Typography
- Choose from available font families
- Adjust font sizes for different elements
- Set font weights and styles

#### Layout Options
- Select sidebar positions
- Choose content width
- Configure archive page layouts

#### Additional CSS
- Add custom CSS for advanced customization

### Widgets

Origamiez includes multiple widget areas to display content:

1. Go to **Appearance > Widgets**
2. Drag and drop widgets to the following areas:
   - Main Top
   - Main Center Top
   - Main Center Left
   - Main Center Right
   - Main Center Bottom
   - Main Bottom
   - Left Sidebar
   - Right Sidebar
   - Footer (1-5)

### Featured Content

To set up featured content on your homepage:

1. Create posts with featured images
2. Assign categories to your posts
3. Go to **Appearance > Widgets**
4. Add the "Origamiez: Posts Slider" widget to a widget area
5. Configure the widget settings to display posts from specific categories

## Special Features

### Post Formats

Origamiez supports various post formats:

1. When creating a post, look for the "Format" box in the right sidebar
2. Choose from:
   - Standard (default)
   - Gallery (for image galleries)
   - Video (for video content)
   - Audio (for audio content)

### Custom Templates

Origamiez includes several page templates:

- **Full Width**: Displays content without sidebars
- **Full Width Centered**: Displays centered content without sidebars
- **Magazine**: Displays content in a magazine-style layout
- **Three Columns**: Displays content with two sidebars

To use a template:
1. Edit a page
2. In the Page Attributes section, select a template from the dropdown
3. Update the page

### Social Media Integration

To add social media links:

1. Go to **Appearance > Customize**
2. Navigate to the Social Media section
3. Enter your social media profile URLs
4. Save changes

## Plugin Compatibility

Origamiez is compatible with popular WordPress plugins:

### WooCommerce

For e-commerce functionality:
1. Install and activate the WooCommerce plugin

3. Origamiez will automatically apply styling to your shop pages

### bbPress

For forum functionality:

For Q&A functionality:
1. Install and activate the DW Question & Answer plugin

3. Origamiez will automatically apply styling to your Q&A pages

## Mobile Responsiveness

Origamiez is fully responsive and will adapt to different screen sizes. The theme includes:

- Mobile-friendly navigation menu
- Responsive images and galleries
- Optimized layouts for tablets and smartphones

## Performance Optimization

To ensure optimal performance:

1. Use optimized images (compress before uploading)
2. Install a caching plugin
3. Use a content delivery network (CDN) if possible
4. Limit the number of widgets in high-traffic areas

## Troubleshooting

### Common Issues

1. **Menu items not appearing**
   - Ensure you've assigned the menu to a location
   - Check that menu items are published and visible

2. **Widgets not displaying**
   - Verify widgets are added to the correct widget areas
   - Check that widget content is properly configured

3. **Featured images not showing**
   - Ensure featured images are large enough (minimum 800x600px recommended)
   - Check that the post format supports featured images

4. **Layout appears broken**
   - Clear your browser cache
   - Disable conflicting plugins
   - Contact theme support if issues persist

## Updates

To update the theme:

1. Back up your website
2. Go to **Appearance > Themes**
3. If an update is available, you'll see a notification
4. Click **Update Now**

Alternatively, you can manually update by uploading the new version through the WordPress admin.
